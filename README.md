## 1. 安装

```bash
# 安装
npm install -g @vue/cli

# 升级
npm update -g @vue/cli

# 查看版本
vue --version
```

## 2. 配置

```bash
# 创建项目
vue create my-project

# 启动项目
cd my-project
npm run serve
```

## 3. 运行

```bash
# 运行项目
npm run serve

# 打包项目
npm run build
```
